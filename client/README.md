# React Beer Ecommerce Challenge

This is the client for the Beer Ecommerce Challenge.

## Requirements

- Node.js
- npm
- Vite

## Installation

```bash
npm install
```

## Running the app

```bash
npm run dev
```

## Testing

```bash
npm run test
```

## Linting

```bash
npm run lint
``` 

## Preview

```bash
npm run preview
``` 

## Build

```bash
npm run build
```
